🎓 PREGUNTAS POSIBLES EN DEFENSA
═══════════════════════════════════════════════════════════════════════════

¿Cómo integran Python con Prolog?
  → pyswip permite consultas desde Python al motor Prolog
  → Conversión de tipos Python → Prolog
  → Manejo de resultados

¿Cuál es la ventaja de usar Prolog?
  → Lógica declarativa (no procedural)
  → Backtracking automático
  → Base de conocimiento centralizada
  → Fácil de extender

¿Cómo ordenan por relevancia?
  → Prolog cuenta coincidencias de síntomas
  → Usa sort/3 para ordenar
  → Retorna diagnósticos ordenados

¿Qué pasa si Prolog no está instalado?
  → Mostrar fallback a síntomas hardcodeados
  → Mensaje de error con instrucciones
  → Sistema sigue parcialmente funcional

Demuestre una consulta Prolog
  → Mostrar ?- diagnosticos_ordenados([...], D)
  → Explicar las reglas aplicadas
  → Mostrar resultado ordenado
